# dept-quiz

## Project setup
```
npm install
```

## Development

### Run node.js server with hot-reloads
```
npm run express
```

### Run vue app with hot-reloads
```
npm run serve
```

## Production

### Deployment
```
npm run build

git push heroku main
```

### Heroku uses the following command to start both front and backend
```
npm run start
```
